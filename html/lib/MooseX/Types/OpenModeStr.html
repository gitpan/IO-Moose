NAME

MooseX::Types::OpenModeStr - Type for open mode string

Back to Top

SYNOPSIS

  package My::Class;
  use Moose;
  use MooseX::Types::OpenModeStr;
  has file => ( isa => 'Str' );
  has mode => ( isa => 'OpenModeStr' );

  package main;
  my $fin  = My::Class->new( file => '/etc/passwd', mode => 'r' );
  my $fout = My::Class->new( file => '/tmp/pwdnew', mode => 'w' );

Back to Top

DESCRIPTION

This module provides Moose type which represents open mode string. It can be Perl-style canonical mode string (i.e. "+>") or C-style mode string (i.e. "w+").
